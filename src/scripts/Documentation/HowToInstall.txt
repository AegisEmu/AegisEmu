How to install ArcScripts?
Windows:

1: Compile ArcEmu (ULTIMATE REV) with C++ (2008 or last)
2: Compile your ArcScripts with C++ (2008 or last)
3: Copy all .LUA files in the LUA Scripting foulder to your script_bin foulder.
4: Import to your world database the ArcScripts_Full.sql
5: Set up your server and HAVE FUN!
